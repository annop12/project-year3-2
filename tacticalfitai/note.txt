Speed (ความเร็ว)
คำนวณจาก: Progressive Carries (การพาบอลเดินหน้า) + Successful Dribbles
สูตร: 75 + (prog_carr_percentile * 15) - (pass_norm * 5)
นักเตะที่พาบอลเยอะและผ่านคนได้ดี = Speed สูง

Strength (พลัง)
คำนวณจาก: Aerial Duel Win % (เปอร์เซ็นต์การชนะดวลอากาศ)
สูตร: 60 + (aerial_pct - 30) * 0.8
ชนะดวลหัวได้เยอะ = Strength สูง

Passing (การส่งบอล)
ใช้จาก: Pass Completion % (เปอร์เซ็นต์ส่งบอลสำเร็จ)
ข้อมูลจริงจาก FBref โดยตรง (60-95%)

Vision (วิสัยทัศน์)
คำนวณจาก: Progressive Passes (การส่งบอลเดินหน้า) + Key 
Passes (การส่งบอลสำคัญ)
สูตร: 68 + (prog_pass + key_pass * 0.8) * 3.5
ส่งบอลเดินหน้าและสร้างโอกาสเยอะ = Vision สูง

Aggression (ความก้าวร้าว)
คำนวณจาก: Fouls Committed (ฟาวล์) + Yellow Cards (ใบเหลือง)
สูตร: 68 + (fouls + yellows) * 8

Composure (ความสงบ)
คำนวณจาก: Goals vs xG (ทำประตูได้มากกว่าที่คาดหวังหรือไม่)
สูตร: 78 + (goals_total - xg_total) * 3
ยิงได้มากกว่า xG = Composure สูง (เย็นใจตอนยิง)

OffTheBall (การเคลื่อนไหวนอกบอล)
คำนวณจาก: Progressive Carries (การพาบอลเดินหน้า)
สูตร: 70 + (prog_carr * 3.8)


🔧 การปรับแต่งเพิ่มเติม (จาก fix_attributes.py)
หลังจากได้ข้อมูลมาแล้ว มีการปรับแต่งเพิ่มเติมใน 2 ค่า:
PressActions (การกดดัน): คำนวณใหม่จาก Speed + OffTheBall
สูตร: 7.5 + (speed_factor * 3) + (offball_factor * 2)
Aggression: ปรับแต่งใหม่จาก Strength + PressActions
สูตร: 75 + (strength_factor * 8) + (press_factor * 5)
